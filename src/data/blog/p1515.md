💰 Catégorie Budget : Démarrer sans se ruiner
--------------------------------------------

### GMKtec NucBox M5 Plus – 340-675€

**Le champion du rapport qualité/prix 2025**

- **Processeur** : AMD Ryzen 7 5825U (8 cœurs/16 threads, jusqu’à 4.5 GHz)
- **RAM** : 16-32 GB DDR4 (upgradable 64 GB)
- **Stockage** : 512 GB – 1 TB NVMe (2 slots M.2)
- **Réseau** : Dual 2.5 GbE + WiFi 6E
- **Consommation** : 10W idle, 35W full load

**Pourquoi lui ?**  
C’est le sweet spot absolu pour un homelab débutant. Le Ryzen 7 5825U est 20% plus performant que le 5700U, avec des cores uniformes (pas de galère P/E cores). Les **2 ports 2.5 GbE** sont un énorme plus pour isoler le trafic (LAN/WAN séparés ou pfsense).

Assez puissant pour faire tourner 10-15 containers Docker simultanément, silencieux, et il consomme moins qu’une ampoule LED. Parfait pour débuter avec Nextcloud + Jellyfin + Vaultwarden sans craindre la facture électrique.

**Idéal pour** : Premier homelab, budget limité, usage 24/7  
**Limite** : Pas de support ECC RAM, architecture Zen 3+ (pas Zen 4)

**Le monstre de puissance**

- **RAM** : Jusqu’à 96 GB DDR5 (support ECC avec certaines barrettes)

Le slot PCIe x16 ouvre des possibilités de dingue :

- Carte réseau 4x 10 GbE
- GPU low-profile pour du calcul AI local
- Carte HBA pour brancher 8+ disques durs

**Benchmark** : 12 VMs (dont 3 Windows Server) + 30 containers → CPU à 60%, 48 GB RAM utilisés, température stable à 65°C.

**Idéal pour** : Cluster homelab, virtualisation intensive, lab certif pro  
**Limite** : Prix élevé (surtout configuré), overkill pour un usage basique, consommation ~25W idle

**Idéal pour** : Fans Apple, usage macOS, containers Docker simples

⚠️ Les pièges à éviter
----------------------

### ❌ Acheter un Raspberry Pi pour un homelab

Les Raspberry Pi, c’est génial… pour apprendre. Mais en 2025, un mini PC x86 à 340€ (GMKtec M5 Plus) est :

- 3x plus puissant
- Compatible avec TOUS les logiciels (pas de galère ARM)
- Évolutif (tu peux upgrade RAM et stockage)

**Exception** : Si tu veux juste Pi-hole ou un petit serveur DNS, un Raspberry Pi 4/5 suffit largement.

### ❌ Négliger le réseau

Un mini PC avec du Gigabit Ethernet (1 GbE), c’est bien. Mais en 2025, privilégie du **2.5 GbE minimum**. Ça change tout pour :

- Transférer tes fichiers NAS rapidement
- Streamer plusieurs flux Jellyfin 4K simultanés
- Backups rapides

**Bonus** : Les modèles avec 2 ports réseau (GMKtec M5 Plus, MS-A2) permettent d’isoler le trafic (LAN/WAN séparés, pfsense/OPNsense).

### ❌ Sous-estimer la consommation électrique

Un mini PC qui tourne 24/7 (prix électricité 0.25€/kWh) :

- **10W** = ~22€/an
- **30W** = ~66€/an
- **100W** (vieux serveur) = ~220€/an

Sur 5 ans, la différence entre un mini PC efficient (10W) et un vieux serveur (100W) = **990€** d’électricité. C’est pas rien.

💬 Questions fréquentes
----------------------

### Est-ce que je peux virtualiser Windows sur un mini PC ?

Oui, sans problème. Avec 32 GB de RAM, tu peux faire tourner 2-3 VMs Windows simultanément. Utilise Proxmox ou VMware ESXi (gratuit pour usage perso).

### Quel stockage prévoir pour un NAS personnel ?

Compte **500 GB par personne** pour les documents, puis ajoute selon tes besoins :

- Photos/vidéos iPhone : 100-200 GB/an
- Blu-ray rippés : 25-50 GB par film
- Musique FLAC : 50 GB pour 500 albums

Mon conseil : 1 TB NVMe pour l’OS + services, puis un SSD externe USB-C 4 TB pour les médias.

### C’est légal d’auto-héberger en France ?

Oui, totalement. Tu peux héberger ce que tu veux chez toi, tant que ce n’est pas illégal (piratage, contenu pédopornographique, etc.). Pour les films/séries sur Jellyfin, seuls tes propres Blu-ray/DVD rippés sont autorisés.

### Mon FAI va me bloquer si j’héberge ?

Non. Les FAI français (Orange, Free, SFR, Bouygues) autorisent l’auto-hébergement. Par contre :

- Ouvre les ports nécessaires sur ta box
- Utilise un nom de domaine (OVH, Gandi : ~5€/an)
- Active le DynDNS si tu n’as pas d’IP fixe

### Combien de temps pour setup un homelab complet ?

- **Setup basique** (OS + Docker) : 1-2h
- **Premiers services** (Nextcloud + Jellyfin) : 3-4h
- **Sécurisation** (HTTPS, pare-feu, monitoring) : 2-3h

**Total** : Un week-end tranquille suffit pour avoir un homelab opérationnel.
