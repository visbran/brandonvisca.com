## Vue d'Ensemble : Les 3 Launchers en Bref

### Spotlight (natif macOS)

**Créé** : 2005 (macOS Tiger)  
**Créateur** : Apple  
**Prix** : Gratuit (inclus macOS)  
**Philosophie** : Simple, intégré, universel

**Forces** :
- ✅ Préinstallé
- ✅ Rapide et léger
- ✅ Intégration native macOS
- ✅ Recherche universelle (apps, fichiers, emails, contacts)

**Faiblesses** :
- ❌ Features limitées
- ❌ Pas de customisation
- ❌ Pas de clipboard history
- ❌ Pas de snippets
- ❌ Pas de workflows

---

### Alfred (le vétéran)

**Créé** : 2010  
**Créateurs** : Andrew & Vero Pepperell (UK)  
**Prix** : Gratuit (limité) / Powerpack 45€ (one-time) / Mega Support 78€ (lifetime updates)  
**Philosophie** : Vitesse, précision, workflows custom

**Forces** :
- ✅ **Vitesse** : Le plus rapide des 3
- ✅ **Workflows** : Automation puissante (Python, Bash, AppleScript)
- ✅ **Recherche fichiers** : Meilleure que Raycast et Spotlight
- ✅ **Stabilité** : 15 ans de développement
- ✅ **Une seule interface** : Sentence-based (pas de menus)

**Faiblesses** :
- ❌ **Payant** : Powerpack requis pour clipboard, snippets, workflows
- ❌ **UI vieillissante** : Design 2010
- ❌ **Pas d'AI** : Pas d'intégration ChatGPT/Claude native
- ❌ **Extensions compliquées** : Workflows = code requis

---

### Raycast (le challenger)

**Créé** : 2020  
**Créateurs** : Thomas Paul Mann & Peter Nikolaev (ex-Facebook engineers)  
**Prix** : Gratuit (généreux) / Pro 8$ ou 96$/an  
**Philosophie** : Moderne, extensible, tout-en-un

**Forces** :
- ✅ **UI moderne** : Design 2025, polie
- ✅ **Gratuit** : Clipboard, snippets, window management inclus gratuitement
- ✅ **Extensions** : Store avec 1000+ extensions
- ✅ **AI intégré** : ChatGPT, Claude (Pro)
- ✅ **Window management** : Remplace Rectangle gratuitement
- ✅ **Calendrier** : Intégration meeting (remplace Fantastical)

**Faiblesses** :
- ❌ **Vitesse** : Légèrement plus lent qu'Alfred (~0.5s vs 0.3s)
- ❌ **Recherche fichiers** : Inférieure à Alfred
- ❌ **Menu-based** : Pas sentence-based comme Alfred (courbe apprentissage)
- ❌ **Pro payant** : AI, cloud sync, themes = abonnement

## Migration : Comment Passer de X à Y

### De Spotlight à Alfred

**Durée** : 1 jour

**Étapes** :
```
1. Installe Alfred
2. Change hotkey : Cmd+Space → Alfred
3. Désactive Spotlight hotkey (Réglages > Spotlight)
4. Teste recherches pendant 1 journée
5. Achète Powerpack si convaincu (45€)
```

**Courbe apprentissage** : Faible (interface similaire)

---

### De Spotlight à Raycast

**Durée** : 2 jours

**Étapes** :
```
1. Installe Raycast
2. Setup : Importer calendriers, config extensions
3. Change hotkey : Cmd+Space → Raycast
4. Explore extensions (Calendrier, Spotify, etc.)
5. Configure window management (remplace Rectangle)
```

**Courbe apprentissage** : Moyenne (menu-based vs sentence-based)

---

### D'Alfred à Raycast

**Durée** : 1 semaine

**Étapes** :
```
1. Export snippets Alfred → Import Raycast
2. Identifie workflows critiques → Trouve équivalents Raycast
3. Configure hotkeys similaires
4. Teste recherche fichiers (possiblement inférieure)
5. Décide : keep Alfred ou switch complet
```

**Difficultés** :
- ⚠️ Recherche fichiers moins bonne dans Raycast
- ⚠️ Workflows complexes : pas toujours d'équivalent
- ⚠️ Vitesse légèrement inférieure

**Mon conseil** : Utilise les **deux en parallèle** 1 semaine avant décider.

---

### De Raycast à Alfred

**Durée** : 1 semaine

**Étapes** :
```
1. Achète Alfred Powerpack (45€)
2. Import snippets Raycast → Alfred
3. Réapprends sentence-based (vs menu-based)
4. Trouve équivalents extensions → Workflows
5. Accepte perte window management (réinstalle Rectangle)
```

**Gains** :
- ✅ Vitesse +30%
- ✅ Recherche fichiers meilleure
- ✅ Workflows plus puissants

**Pertes** :

### Semaine 3-4 : Alfred Powerpack (78€ lifetime)

**Ressenti** : **Puissant, rapide, stable**.

**Ce que j'aime** :
- ✅ Recherche fichiers **ultra-rapide** (critical pour mon workflow dev)
- ✅ Snippets : `;email`, `;sig`, `;addr` → Gain 5 min/jour
- ✅ Clipboard history : Je ne perds plus jamais rien
- ✅ Workflows : Compression images, upload S3, git commits

**Ce qui manque** :

**Conclusion** : **Excellent pour dev/sysadmin**. Worth 78€ si tu codes quotidiennement.

**Ressenti** : **Moderne, tout-en-un, mais légèrement plus lent**.

- ✅ Window management gratuit (j'ai viré Rectangle)
- ✅ Calendrier meetings (killer feature)
- ✅ Clipboard history spacieuse (pin items)

**Ce qui manque** :
- ❌ Recherche fichiers **inférieure** à Alfred (critical pour moi)

**Pro (96€/an)** :

**Conclusion** : **Excellent all-in-one** si recherche fichiers pas critique.
