# Cloudflare Pages — Configuration

## Dashboard settings

Go to: **Workers & Pages → brandonvisca.com → Settings → Build & deployments**

| Setting | Value |
|---|---|
| Build command | `pnpm run build:cf` |
| Build output directory | `dist` |
| Root directory | *(leave empty)* |

## Environment variables (required)

| Variable | Value | Environment |
|---|---|---|
| `NODE_VERSION` | `20` | Production + Preview |
| `PNPM_VERSION` | `10` | Production + Preview |

## Why `build:cf` instead of `build`

The default `build` script includes `astro check` (TypeScript type-checking) which can timeout on CF Pages free tier.
`build:cf` skips it: `astro build && pagefind --site dist`

## Deployment trigger

Every push to `main` → automatic deployment.
Branch `preview/*` → preview deployment (separate URL).
